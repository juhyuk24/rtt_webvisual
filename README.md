# 노드, 링크 텍스트 데이터를 읽어서 화면에 출력

# 노드 데이터
Node_id (노드 안에서 ID 1, 2, 3, 4)

Node_guid (노드를 식별할 수 있는 유일한 ID)

Category_name (넘어온 데이터, 용도는 모름)

Family_name (넘어온 데이터, 용도는 모름)
